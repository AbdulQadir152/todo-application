<h1>Todo Application</h1>

<p>
A full-stack Todo Application starting with a database-first design and evolving incrementally into a complete system using Django and modern frontend technologies.
</p>

<h2>Project Status</h2>
<p>
Currently focused on database design and refactoring. Backend and frontend will be added step by step as learning progresses.
</p>

<h2>Tech Stack (Planned)</h2>
<ul>
  <li>Database: MySQL</li>
  <li>Backend: Django</li>
  <li>Frontend: HTML, CSS, JavaScript / Next.js</li>
</ul>
